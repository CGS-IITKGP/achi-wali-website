Database layer

Models, relationships, and repository utilities.

Data flow

Services call repositories which wrap Mongoose models. The generic repository provides CRUD, paginated queries, and transactional helpers.

Models and relationships

Model Key fields Relationships
User email, roles, designation, profileImgUrl Belongs to Team via teamId
Team name, description, members Members reference User ids
Project portfolio, title, links, media Author + collaborators reference User ids
Blog title, slug, tags, content Author + collaborators reference User ids
Featured contentType, contentId, isHighlight References Blog or Project via contentId
Media key, url, sizeBytes, resourceType uploadedBy references User id
SignUpRequest email, otpHash, expiresAt TTL index removes expired requests

Transactions and sessions

  • connectToDatabase() caches the connection in global.mongooseCache.
  • withSession() wraps multi-step updates in a transaction.
  • Repositories accept an optional ClientSession.
