API routes

Routes, inputs, and authorization levels across the backend.

Auth

Method Path Auth Notes
POST /api/auth/sign-in Public Sets session cookie on success.
POST /api/auth/sign-out Public Overwrites session cookie.
POST /api/auth/sign-up Public Deprecated OTP sign-up flow.
PATCH /api/auth/change-password Session Requires a valid session cookie.
GET /api/auth/me Session Returns the current user summary.
POST /api/auth/refresh-session Session Rotates the JWT cookie.
GET /api/auth/google Public Redirects to Google OAuth consent.
GET /api/auth/google/callback Public Exchanges code for session and redirects.

Examples

POST /api/auth/sign-in
{
  "email": "user@cgs.example",
  "password": "secret"
}

200 OK
{
  "action": true,
  "data": {}
}

POST /api/auth/refresh-session
{}

200 OK
{
  "action": true,
  "data": {}
}
Sign-in and refresh-session set the session cookie in the response headers.

Blog

GET /api/blog?target=all Public Returns a list view of all blogs.
GET /api/blog?target=my Session Returns blogs authored by the session user.
GET /api/blog/view/:slug Public Full blog content by slug.

PATCH /api/blog/:id Admin or author Updates title, slug, content, tags, or cover image.

Method Path Auth Notes

Media

POST /api/media/sign Session Returns a Cloudinary upload signature.
POST /api/media Member Stores uploaded media metadata.
DELETE /api/media/:id Admin or uploader Deletes media and invalidates Cloudinary asset.

Examples

{
  "action": true,
